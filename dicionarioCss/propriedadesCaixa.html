Propriedades de Caixa (Box Model)

As propriedades de caixa são usadas para controlar o layout e o espaçamento dos elementos em uma página. Aqui estão algumas das principais propriedades de caixa:

  • width - Define a largura do elemento.
  • height - Define a altura do elemento.
  • margin - Define a margem externa do elemento.
  • padding - Define o preenchimento interno do elemento.
  • border - Define a borda do elemento.
  • box-sizing - Controla como as dimensões do box são calculadas.
  • display - Controla como o elemento é exibido (block, inline, flex, etc.).
  • overflow - Controla o que acontece quando o conteúdo excede as dimensões do box.
  • position - Controla o posicionamento do elemento (static, relative, absolute, fixed, sticky).
  • z-index - Controla a sobreposição de elementos (quem fica em cima de quem).
  • background - Controla o fundo do elemento.
  • border-radius - Controla o arredondamento das bordas do elemento.
  • box-shadow - Controla a sombra do elemento.
