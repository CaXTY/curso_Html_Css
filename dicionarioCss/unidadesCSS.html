Unidades CSS (Units)

As unidades CSS são usadas para definir tamanhos, espaçamentos e outras dimensões em um documento HTML. Aqui estão algumas das principais unidades CSS:

  • px - Pixels, uma unidade fixa comumente usada para definir tamanhos de elementos.
  • em - Uma unidade relativa que é baseada no tamanho da fonte do elemento pai.
  • rem - Uma unidade relativa que é baseada no tamanho da fonte do elemento raiz (html).
  • % - Porcentagem, uma unidade relativa que é baseada no tamanho do elemento pai.
  • vh - Viewport Height, 1vh é igual a 1% da altura da viewport.
  • vw - Viewport Width, 1vw é igual a 1% da largura da viewport.
  • vmin - Viewport Minimum, 1vmin é igual a 1% da menor dimensão da viewport.
  • vmax - Viewport Maximum, 1vmax é igual a 1% da maior dimensão da viewport.
