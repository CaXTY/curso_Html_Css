Pseudo-classes

As pseudo-classes servem para criar efeitos quando o usuário interage com um elemento, como :hover ou :focus.

  • :hover - Aplica estilos quando o mouse está sobre o elemento.
  • :focus - Aplica estilos quando o elemento está em foco.
  • :active - Aplica estilos quando o elemento está sendo ativado.
  • :checked - Seleciona um elemento que está marcado.
  • :disabled - Seleciona um elemento que está desativado.
  • :first-child - Seleciona o primeiro filho de um elemento pai.
  • :last-child - Seleciona o último filho de um elemento pai.
  • :nth-child(n) - Seleciona o enésimo filho de um elemento pai.
