Seletores Básicos

Os seletores básicos são fundamentais para aplicar estilos a elementos HTML. Eles permitem que você selecione elementos com base em suas tags, classes ou IDs.

  • * {} - seleciona todos os elementos.
  • #id {} - Seleciona todos os elementos com um ID específico.
  • .class {} - Seleciona todos os elementos com uma classe específica.
  • h1, h2 {} - Seleciona vários elementos juntos.
  • section p {} - Seleciona p dentro de section.
  • section > p {} - Seleciona p que são filhos diretos de section.
  • li + li {} - Seleciona o primeiro li após outro li.
  • h1 ~ p {} - Seleciona todos os p após um h1.
